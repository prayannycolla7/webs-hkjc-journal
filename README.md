# webs-hkjc-journal

## 项目简介

本仓库用于归档与发布一系列独立 HTML 页面。这些页面以静态网页形式存储，可直接通过浏览器访问，无需后端服务支持。仓库内容不针对任何特定域名或网站，仅作为页面资料的集中存放与版本管理用途。

## 目录说明

```
.
├── index.html          # 仓库入口页面（如有）
├── pages/              # 存放主要 HTML 页面
├── assets/             # 页面依赖的静态资源（CSS、JS、图片等）
└── README.md           # 本文件
```

- `pages/`：按时间或主题整理的独立 HTML 页面文件夹。
- `assets/`：页面所需的样式、脚本、图像等资源文件，按需组织。

## 页面归档说明

仓库中的 HTML 页面均为独立完整文档，彼此之间无强关联。每个页面可单独打开并正常显示。归档内容可能涵盖不同时间、不同主题的静态页面，不保证页面间内容一致性或连续性。

页面更新采用 Git 版本控制，历史记录可追溯每次变更。如需检索特定页面，可通过提交日志或目录结构查找。
